## Reverse String

Question:

> 反转字符串

**For example**
```
Input：["h","e","l","l","o"]
Output：["o","l","l","e","h"]
```
**Note**
* 原地修改
* 空间复杂度O(1)
* * *
### java (双指针、前后交换)
```java
class Solution {
  public void reverseString(char[] s) {
    for(int i = 0; i<s.length/2; i++) {
      char temp = s[i];
      s[i] = s[s.length-i-1];
      s[s.length-i-1] = temp;
    }
  }
}
```
### python 3 (一行代码)
```python
class Solution:
  def reverString(self, s):
    s.reverse()
```

### Java （双指针）
```java
class Solution {
     public void reverseString(char[] s) {
        if(s == null || s.length <= 0 ){
            return;
        }
        for (int i = 0,j = s.length-1 ; i < j ; i++,j--){
            char  temp = s[i];
            s[i] = s[j];
            s[j] = temp;
        }
    }
}
```
> 内存占用和执行效率与前一种差不了多少
