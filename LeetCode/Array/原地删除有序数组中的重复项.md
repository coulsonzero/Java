
`原地删除排序数组中的重复项，返回新长度`

```
输入：nums = [0,0,1,1,1,2,2,3,3,4]
输出：5
nums = [0,1,2,3,4]
```

### java
```java
public static int removeDuplicates(int[] nums) {
    if(nums.length == 0) return 0;
	
    int i = 0;
    for (int j = 1; j < nums.length; j++) {
        if (nums[i] != nums[j]) {
            i++;
            nums[i] = nums[j];
        }
    }
    return i+1; 
}
```
### python3
```python3
class Solution:
    def removeDuplicates(self, nums):

        for i in range(len(nums)-1, -1, -1):    # 比较最后两项是否重复
            if nums[i] == nums[i-1]:
                del nums[i]
        return len(nums)
```
